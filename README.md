# NACA 4-Digit Airfoil Generator

A Python implementation of the NACA 4-digit airfoil equations using NumPy and Matplotlib.

## Features
- Generates any NACA 4-digit airfoil
- Computes camber line and thickness distribution
- Plots upper and lower surfaces
- Uses numerical differentiation for surface orientation

## Libraries
- NumPy
- Matplotlib
